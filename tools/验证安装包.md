# 文件完整性校验说明

本目录提供用于验证安装包是否为官方原版的工具。

---

## 📌 使用方法（推荐）

1. 将以下文件放在同一目录：
   - 安装包（.exe 或 .msi）
   - check.ps1
   - check.bat

2. 双击运行 `check.bat`

3. 根据提示查看结果：

   - 显示 **“OK (official file).”** ✅ → 可以正常使用
   - 显示 **“WARNING (NOT official) !!!”** ❌ → 文件可能已被修改，请不要使用

---

## 🔍 校验原理（简单说明）

本工具会对安装包计算 SHA256 值，并与官方发布的值进行比对。

只要有任何改动（哪怕改一个字节），结果都会不同。

---

## ⚠️ 注意事项

- 请确保安装包来源可靠（建议从官方或 GitHub 下载）
- 第三方下载站提供的文件，可能被修改
- 如果校验结果为“警告”，请立即删除该文件
- 请从 RobustLuo 的 GitHub Releases 下载：https://github.com/RobustLuo/light-c/releases

---

## ⚠️ 关于第三方下载来源

我已发现：部分第三方下载站提供的安装包，其 SHA256 与官方发布版本不一致。

这意味着文件内容已经发生变化。

由于无法确认这些修改的具体内容及来源，存在潜在安全风险。

👉 强烈建议仅从官方渠道下载，并使用本工具进行校验。

---

## 💡 提示

如果不想进行校验，最简单安全的方式是：

👉 仅从官方发布渠道下载使用
